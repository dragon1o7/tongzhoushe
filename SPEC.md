# 同舟社社群网站 - 规格文档 (SPEC.md)

## 项目概述

**项目名称**: 同舟社社群网站 (Tongzhou Community)

**项目定位**: 为同舟社社群成员打造的综合性社区平台，提供交流互动、知识分享、个人展示等功能。

**技术栈**:
- 前端框架：Next.js 14+ (App Router)
- UI 组件：shadcn/ui + Tailwind CSS
- 数据库：SQLite (开发) / PostgreSQL (生产)
- ORM: Prisma
- 认证：NextAuth.js
- 部署：Vercel

**核心功能**:
1. 用户认证系统 (手机号/密码登录)
2. 社区广场 (发帖、评论、互动)
3. AI 学院 (文章发布与阅读)
4. 个人主页 (用户信息展示)

## 数据模型概述

### 核心实体

1. **User (用户)**
   - 基础信息：姓名、邮箱、手机号、头像、简介
   - 关联：发布的帖子、评论

2. **Category (板块)**
   - 板块信息：名称、描述、图标
   - 关联：帖子列表

3. **Post (帖子)**
   - 内容信息：标题、内容、标签
   - 统计：浏览量、点赞数
   - 关联：作者、板块、评论

4. **Comment (评论)**
   - 评论内容
   - 关联：帖子、作者

5. **Article (文章)**
   - 文章内容：标题、内容、分类
   - 统计：浏览量
   - 发布时间

### 关系图

```
User ──< Post >── Category
  │        │
  │        └──< Comment
  └──────────────^

Article (独立实体)
```
